## 介绍

《WINGFLEXSIM 设备接入开发文档》可以帮助你，在你的软件中接入 WINGFLEXSIM 设备。为了方便开发者使用自己熟悉的编程语言进行接入，我们目前提供 JavaScript 语言的示例代码。你可以很轻松地在HTML中编写测试代码，在浏览器中验证接入效果。

在理解了接入方法后，你可以使用几乎任何语言框架来接入 WINGFLEXSIM 设备，整合到任何连接软件上。

目前，我们提供接入文档的设备：

- A320 FCU CUBE

其他设备接入文档仍在编辑和测试中。
